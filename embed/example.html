slim code Embed Examples

This page demonstrates how to use the slim code embed mode with various query parameters. You can embed code snippets from files, GitHub repositories, or GitHub Gists with a customizable editor view.

Query Parameters Reference

Parameter Description Default Example
path Path to the file to display - path=js/main.js
content Direct code content (URL encoded) - content=console.log(%22Hello%22)
lang Language for syntax highlighting Detected from file extension lang=javascript
theme Editor theme vs-dark theme=vs-light
title Title to display in header Filename or "Code Snippet" title=My%20Code
header Show/hide header true header=false
footer Show/hide footer true footer=false
readOnly Make editor read-only true readOnly=false
fontSize Font size in pixels 14 fontSize=16
tabSize Tab size in spaces 4 tabSize=2
wordWrap Enable word wrapping false wordWrap=true
lineNumbers Show line numbers true lineNumbers=false
minimap Show minimap false minimap=true
highlight Highlight specific lines - highlight=1,3-5,7
startLine First line to display 1 startLine=10
endLine Last line to display - endLine=20
source Source type file source=github
githubUser GitHub username - githubUser=microsoft
githubRepo GitHub repository name - githubRepo=vscode
githubPath Path to file in GitHub repo - githubPath=README.md
branch Branch name for GitHub repo main branch=develop
gistId GitHub Gist ID - gistId=abc123
gistFile Filename in Gist (if multiple files) First file gistFile=example.js

Implementation Guide

How to Use

To embed a code snippet on your website, use an iframe with the src attribute pointing to the slim code embed URL with your desired parameters:

<iframe src="https://your-domain.com/slim-code/embed/index.html?path=your-file-path.js" width="100%" height="400" frameborder="0" ></iframe>

You can combine multiple parameters using the & character:

<iframe src="https://your-domain.com/slim-code/embed/index.html?path=your-file-path.js&theme=vs-dark&highlight=1,5-10&readOnly=true" width="100%" height="400" frameborder="0" ></iframe>

Advanced Usage

For more complex scenarios, you can use the source parameter to specify where the code should be loaded from:

  • file: Load from a file path (default)
  • content: Use direct content provided in the URL
  • github: Load from a GitHub repository
  • gist: Load from a GitHub Gist

Example for loading from GitHub:

<iframe src="https://your-domain.com/slim-code/embed/index.html?source=github&githubUser=username&githubRepo=repo-name&githubPath=path/to/file.js" width="100%" height="400" frameborder="0" ></iframe>
